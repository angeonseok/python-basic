## 모듈 (Module)

> **모듈** : 함수, 변수, 클래스 등을 모아 둔 **하나의 `.py` 파일**. 코드를 분리해 재사용하고 관리하기 쉽게 만든다.

### 모듈 활용

> `import`로 모듈을 불러와 그 안의 기능을 사용한다.

```python
import math

print(math.pi)         # 3.141592653589793
print(math.sqrt(16))   # 4.0
```

| 방식 | 문법 | 사용 예시 |
|------|------|-----------|
| 모듈 전체 | `import math` | `math.sqrt(16)` |
| 특정 기능만 | `from math import sqrt` | `sqrt(16)` |
| 여러 개 | `from math import pi, sqrt` | `pi`, `sqrt(16)` |
| 별칭 | `import math as m` | `m.sqrt(16)` |
| 전체 가져오기 | `from math import *` | `sqrt(16)` |

```python
from math import sqrt, pi
print(sqrt(25))   # 5.0
print(pi)         # 3.141592653589793

import math as m  # 별칭(alias)
print(m.pi)       # 3.141592653589793
```

> ⚠️ `from math import *` 는 어떤 이름이 들어왔는지 추적이 어렵고 **이름 충돌** 위험이 있어 권장하지 않는다.

### 사용자 정의 모듈

> 직접 만든 `.py` 파일도 모듈로 가져와 쓸 수 있다.

```python
# my_module.py
def add(a, b):
    return a + b

PI = 3.14
```

```python
# main.py (같은 폴더)
import my_module

print(my_module.add(3, 5))   # 8
print(my_module.PI)          # 3.14

from my_module import add
print(add(1, 2))             # 3
```

## 파이썬 표준 라이브러리 (Python Standard Library)

> 파이썬 설치 시 **기본 포함**되어, 따로 설치 없이 `import`만으로 쓸 수 있는 모듈 묶음

| 모듈 | 용도 | 예시 |
|------|------|------|
| `math` | 수학 연산 | `math.sqrt(16)` |
| `random` | 난수 생성 | `random.randint(1, 6)` |
| `datetime` | 날짜·시간 | `datetime.date.today()` |
| `os` | 운영체제 기능 | `os.getcwd()` |
| `sys` | 인터프리터 제어 | `sys.argv` |

```python
import random
print(random.randint(1, 6))        # 1~6 중 랜덤 정수
print(random.choice(["a", "b"]))   # 리스트에서 랜덤 선택

import datetime
print(datetime.date.today())       # 오늘 날짜
```

### 패키지 (Package)

> **패키지** : 여러 모듈을 폴더 단위로 묶은 것. 관련 모듈을 계층적으로 관리한다.

```
my_package/
├── __init__.py      # 패키지임을 표시
├── module_a.py
└── module_b.py
```

```python
from my_package import module_a
from my_package.module_b import some_func
```

외부 패키지는 **PyPI**에서 `pip`로 설치한다.

```bash
pip install requests   # 외부 패키지 설치
```

| 구분 | 설명 |
|------|------|
| 표준 라이브러리 | 파이썬에 기본 포함 (설치 불필요) |
| 외부 패키지 | `pip install`로 별도 설치 |

## 제어문 (Control Flow)

> **제어문** : 코드의 **실행 흐름을 조절**하는 문법. 조건에 따라 분기하거나(조건문), 반복 실행한다(반복문).

| 종류 | 키워드 | 역할 |
|------|--------|------|
| 조건문 | `if`, `elif`, `else` | 조건에 따라 분기 |
| 반복문 | `for`, `while` | 코드를 반복 실행 |
| 반복 제어 | `break`, `continue`, `pass` | 반복 흐름 제어 |

---

## 조건문 (Conditional)

### if statement

> 조건이 **참(True)**일 때만 해당 블록을 실행한다.

```python
if 조건:
    조건이_참일_때_실행
elif 다른_조건:
    위가_거짓이고_이게_참일_때
else:
    모두_거짓일_때
```

```python
score = 85

if score >= 90:
    print("A")
elif score >= 80:
    print("B")     # 출력됨
else:
    print("C")
```

| 키워드 | 설명 |
|--------|------|
| `if` | 첫 조건 (필수) |
| `elif` | 추가 조건 (여러 개 가능, 선택) |
| `else` | 모든 조건이 거짓일 때 (선택) |

#### 중첩 조건문

```python
age = 20
if age >= 18:
    if age >= 65:
        print("경로")
    else:
        print("성인")   # 출력됨
```

#### 조건 표현식 (삼항 연산자)

> `참값 if 조건 else 거짓값` — 한 줄로 분기한다.

```python
age = 20
result = "성인" if age >= 18 else "미성년자"
print(result)   # 성인
```

## 반복문 (Loop)

### for statement

> 시퀀스(리스트, 문자열, range 등)의 요소를 **하나씩 순서대로** 꺼내 반복한다.

```python
for 변수 in 반복가능한_객체:
    실행할_코드
```

```python
for fruit in ["apple", "banana"]:
    print(fruit)        # apple → banana

for i in range(3):
    print(i)            # 0 → 1 → 2

for char in "abc":
    print(char)         # a → b → c
```

### while statement

> 조건이 **참인 동안** 계속 반복한다.

```python
while 조건:
    실행할_코드
```

```python
count = 0
while count < 3:
    print(count)        # 0 → 1 → 2
    count += 1          # 종료 조건을 향해 변화시켜야 함
```

> ⚠️ 조건이 계속 참이면 **무한 루프**에 빠진다. 반드시 조건을 변화시키는 코드가 있어야 한다.

| 구분 | for | while |
|------|-----|-------|
| 사용 시점 | 반복 횟수가 정해짐 | 조건에 따라 반복 |
| 대상 | 반복 가능한 객체 | 조건식 |

### 반복 제어

> 반복문의 흐름을 중간에 조절하는 키워드

| 키워드 | 동작 |
|--------|------|
| `break` | 반복문을 **즉시 종료** |
| `continue` | 현재 반복을 건너뛰고 **다음 반복으로** |
| `pass` | 아무것도 하지 않음 (자리만 차지) |

```python
# break: 조건 만족 시 즉시 탈출
for i in range(10):
    if i == 3:
        break
    print(i)        # 0 → 1 → 2

# continue: 짝수만 건너뛰기
for i in range(5):
    if i % 2 == 0:
        continue
    print(i)        # 1 → 3

# pass: 문법상 코드가 필요하지만 실행할 건 없을 때
for i in range(3):
    pass            # 에러 없이 통과
```

### 유용한 내장함수 map & zip

#### map

> 시퀀스의 **모든 요소에 함수를 적용**한 결과를 돌려준다.

```python
nums = ["1", "2", "3"]
result = list(map(int, nums))   # [1, 2, 3]  (문자열 → 정수)

doubled = list(map(lambda x: x * 2, [1, 2, 3]))   # [2, 4, 6]
```

#### zip

> 여러 시퀀스를 **같은 인덱스끼리 묶어** 튜플로 돌려준다.

```python
names = ["건석", "철수"]
ages = [25, 30]

list(zip(names, ages))   # [('건석', 25), ('철수', 30)]

# for문과 함께 자주 사용
for name, age in zip(names, ages):
    print(f"{name}: {age}")   # 건석: 25 → 철수: 30
```

> ⚠️ `map`, `zip`은 결과가 **이터레이터**라서 `list()` 등으로 변환해야 값이 보인다.

## 참고

### 모듈 내부 살펴보기

> 모듈 안에 어떤 기능이 있는지 확인하는 내장 함수

```python
import math

dir(math)        # math의 모든 이름(함수·변수) 목록 반환
help(math.sqrt)  # sqrt 함수의 사용법·설명 출력
```

| 함수 | 용도 |
|------|------|
| `dir(객체)` | 사용 가능한 속성·메서드 목록 |
| `help(객체)` | 상세 설명·사용법 |

### for-else

> 반복문이 `break` **없이 정상 종료**되면 `else` 블록이 실행된다.

```python
for i in range(5):
    if i == 10:
        break
else:
    print("break 없이 끝남")   # 실행됨 (break를 안 만났으므로)
```

```python
# break를 만나면 else는 실행 안 됨
for i in range(5):
    if i == 2:
        break
else:
    print("실행 안됨")
```

> 주로 "찾는 값이 있는지" 검사할 때 유용하다. (찾으면 `break`, 못 찾으면 `else`)

### enumerate

> 반복 시 **인덱스와 값을 함께** 꺼낸다.

```python
fruits = ["apple", "banana", "cherry"]

for index, fruit in enumerate(fruits):
    print(index, fruit)
# 0 apple
# 1 banana
# 2 cherry

# 시작 인덱스 지정 가능
for index, fruit in enumerate(fruits, start=1):
    print(index, fruit)   # 1 apple → 2 banana → 3 cherry
```

> `range(len(fruits))`로 인덱스를 직접 만드는 것보다 `enumerate`가 더 파이썬다운(Pythonic) 방식이다.
