## 데이터 타입 개요

> **타입** : 변수나 값이 가질 수 있는 데이터의 종류  
> **데이터 타입** : 값의 종류와 그 값으로 할 수 있는 '동작(연산)'을 결정하는 속성

| 분류 | 타입 |
|------|------|
| 숫자형 | `int`, `float`, `complex` |
| 텍스트 시퀀스 | `str` |
| 시퀀스 | `list`, `tuple`, `range` |
| 비시퀀스 | `set`, `dict` |
| 기타 | `bool`, `None`, functions |

---

## 숫자형 (Numeric Type)

| 타입 | 설명 | 예시 |
|------|------|------|
| `int` | 소수점 없는 정수 | `42`, `-7` |
| `float` | 소수점 있는 실수 | `3.14`, `-0.5` |
| 지수 표현 | `e` 또는 `E` 사용 | `1.5e3` → `1500.0` |

### 산술 연산자

| 연산자 | 의미 | 예시 | 결과 |
|--------|------|------|------|
| `+` | 덧셈 | `3 + 2` | `5` |
| `-` | 뺄셈 | `3 - 2` | `1` |
| `*` | 곱셈 | `3 * 2` | `6` |
| `/` | 나눗셈 | `7 / 2` | `3.5` |
| `//` | 몫 | `7 // 2` | `3` |
| `%` | 나머지 | `7 % 2` | `1` |
| `**` | 거듭제곱 | `2 ** 3` | `8` |
| `-x` | 음수 부호 | `-5` | `-5` |

### 연산자 우선순위

```
** > -(음수 부호) > *, /, //, % > +, -
```

## 시퀀스 타입 공통 특징

시퀀스 타입이란 **여러 개의 값들을 순서대로 나열하여 저장하는 자료형**이다.

| 특징 | 설명 |
|------|------|
| 순서 | 값들이 순서대로 저장 (정렬 아님) |
| 인덱싱 | 고유 인덱스로 특정 위치의 값 접근/수정 |
| 슬라이싱 | 인덱스 범위로 원하는 부분만 추출 |
| 길이 | `len()` 함수로 값의 개수 확인 |
| 반복 | 반복문으로 값을 하나씩 순서대로 꺼냄 |

## 문자열 (str)

> 문자들의 순서가 있는 **변경 불가능한** 시퀀스 자료형

```python
s1 = 'hello'
s2 = "world"
```

### 이스케이프 시퀀스

| 이스케이프 | 의미 |
|------------|------|
| `\n` | 줄 바꿈 |
| `\t` | 탭 |
| `\\` | 백슬래시 |
| `\'` | 작은따옴표 |
| `\"` | 큰따옴표 |

### f-string

```python
name = "건석"
age = 25
print(f"이름: {name}, 나이: {age}")       # 이름: 건석, 나이: 25
print(f"내년 나이: {age + 1}")            # 내년 나이: 26
```

### 문자열의 불변성

```python
s = "hello"
# s[0] = "H"  ❌ TypeError 발생

s = "H" + s[1:]  # ✅ 새 문자열 생성
print(s)  # Hello
```

## 리스트 (list)

> 여러 개의 값을 순서대로 저장하는 **변경 가능한** 시퀀스 자료형

```python
fruits = ["apple", "banana", "cherry"]
mixed  = [1, "hello", 3.14, True]       # 모든 종류의 데이터 담기 가능
```

### 인덱싱 & 슬라이싱

```python
fruits[0]    # 'apple'
fruits[-1]   # 'cherry'
fruits[0:2]  # ['apple', 'banana']
```

### 가변성 - 값 수정

```python
fruits[0] = "mango"          # 인덱싱으로 단일 수정
fruits[0:2] = ["kiwi", "pear"]  # 슬라이싱으로 여러 값 한번에 변경
```

### 중첩 리스트

```python
matrix = [[1, 2, 3], [4, 5, 6]]
matrix[1][2]  # 6  (인덱스 연달아 사용)
```

## 튜플 (tuple)

> 여러 개의 값을 순서대로 저장하는 **변경 불가능한** 시퀀스 자료형

```python
t1 = (1, 2, 3)
t2 = 1, 2, 3        # 소괄호 생략 가능
t3 = (42,)          # 단일 요소: trailing comma 필수!
t4 = (42)           # ❌ 이건 그냥 int
```

### 튜플은 어디에 쓰이는가?

```python
# 다중 할당
x, y = 10, 20

# 값 교환
x, y = y, x

# 다중 값 반환
def min_max(nums):
    return min(nums), max(nums)

lo, hi = min_max([3, 1, 4, 1, 5])
```

## range

> 연속된 정수 시퀀스를 생성하는 **변경 불가능한** 자료형

```python
range(5)         # 0, 1, 2, 3, 4        (stop만)
range(1, 5)      # 1, 2, 3, 4           (start, stop)
range(0, 10, 2)  # 0, 2, 4, 6, 8       (start, stop, step)
range(5, 0, -1)  # 5, 4, 3, 2, 1       (역순)
```

> ⚠️ `stop` 값은 시퀀스에 **포함되지 않음**

```python
for i in range(3):
    print(i)  # 0, 1, 2
```

## 딕셔너리 (dict)

> key-value 쌍으로 이루어진 **순서 없고 중복 없는, 변경 가능한** 자료형

```python
person = {"name": "건석", "age": 25, "lang": "Python"}
```

### key / value 규칙

| | key | value |
|-|-----|-------|
| 중복 | ❌ 불가 | ✅ 가능 |
| 타입 | 변경 불가능한 타입만 (`str`, `int`, `tuple` 등) | 모든 타입 가능 |

### 값 접근 & 추가 & 변경

```python
person["name"]         # '건석'  (접근)
person["email"] = "a@b.com"  # 추가
person["age"] = 26           # 변경
```

## 세트 (set)

> 순서와 중복이 없는 **변경 가능한** 자료형 — 수학의 집합과 동일

```python
s = {1, 2, 3, 2, 1}
print(s)  # {1, 2, 3}  (중복 자동 제거)
```

### 집합 연산

```python
a = {1, 2, 3}
b = {2, 3, 4}

a | b   # 합집합 → {1, 2, 3, 4}
a - b   # 차집합 → {1}
a & b   # 교집합 → {2, 3}
```

## 형변환 (Type Conversion)

### 암시적 형변환 (파이썬이 자동으로)

```python
print(3 + 1.5)   # 4.5  (int → float 자동 변환)
print(True + 1)  # 2    (bool → int 자동 변환)
```

### 명시적 형변환 (개발자가 직접 지정)

| 함수 | 변환 대상 | 예시 | 결과 |
|------|----------|------|------|
| `int()` | str, float, bool → int | `int("3")` | `3` |
| `float()` | str, int, bool → float | `float("3.14")` | `3.14` |
| `str()` | 모든 타입 → str | `str(42)` | `"42"` |
| `bool()` | 모든 타입 → bool | `bool(0)` | `False` |
| `list()` | 시퀀스 → list | `list("abc")` | `['a','b','c']` |
| `tuple()` | 시퀀스 → tuple | `tuple([1,2])` | `(1, 2)` |
| `set()` | 시퀀스 → set | `set([1,1,2])` | `{1, 2}` |

## 연산자 정리

### 복합 연산자

| 연산자 | 의미 | 예시 | 결과 |
|--------|------|------|------|
| `+=` | 더하고 할당 | `x += 3` | `x = x + 3` |
| `-=` | 빼고 할당 | `x -= 3` | `x = x - 3` |
| `*=` | 곱하고 할당 | `x *= 2` | `x = x * 2` |
| `/=` | 나누고 할당 | `x /= 2` | `x = x / 2` |
| `//=` | 몫 할당 | `x //= 2` | `x = x // 2` |
| `%=` | 나머지 할당 | `x %= 2` | `x = x % 2` |
| `**=` | 거듭제곱 할당 | `x **= 2` | `x = x ** 2` |

### 비교 연산자

| 연산자 | 의미 | 예시 | 결과 |
|--------|------|------|------|
| `==` | 같음 | `3 == 3` | `True` |
| `!=` | 다름 | `3 != 4` | `True` |
| `>` | 초과 | `5 > 3` | `True` |
| `<` | 미만 | `3 < 5` | `True` |
| `>=` | 이상 | `5 >= 5` | `True` |
| `<=` | 이하 | `3 <= 5` | `True` |

### 논리 연산자

| 연산자 | 의미 | 예시 | 결과 |
|--------|------|------|------|
| `and` | 둘 다 True | `True and False` | `False` |
| `or` | 하나라도 True | `True or False` | `True` |
| `not` | 반전 | `not True` | `False` |

**단축 평가 (Short-circuit evaluation)**

```python
# and: 앞이 False면 뒤를 평가하지 않음
False and print("실행 안됨")   # False

# or: 앞이 True면 뒤를 평가하지 않음
True or print("실행 안됨")    # True

# 실용 예시
name = None
result = name or "익명"  # name이 None(falsy)이므로 "익명" 반환
```

### 멤버십 연산자

| 연산자 | 의미 | 예시 | 결과 |
|--------|------|------|------|
| `in` | 포함 여부 | `"a" in "apple"` | `True` |
| `not in` | 미포함 여부 | `5 not in [1, 2, 3]` | `True` |

### 시퀀스형 연산자

```python
[1, 2] + [3, 4]   # [1, 2, 3, 4]   (+: 이어붙이기)
"hi" * 3           # "hihihi"       (*: 반복)
[0] * 5            # [0, 0, 0, 0, 0]
```

### 전체 연산자 우선순위 (높음 → 낮음)

| 우선순위 | 연산자 |
|----------|--------|
| 1 (높음) | `**` |
| 2 | `-(단항)` |
| 3 | `*`, `/`, `//`, `%` |
| 4 | `+`, `-` |
| 5 | `==`, `!=`, `>`, `<`, `>=`, `<=` |
| 6 | `not` |
| 7 | `and` |
| 8 (낮음) | `or` |
